Pydicom - a Python package for handling DICOM files

Pydicom allows easy reading and writing of DICOM files. DICOM is a standard for the transfer of medical images and other medical information between different equipment or software.

Pydicom is released under the terms of the GNU General Public License.

Quick Start

  1. Get and install Python (www.python.org). If you are on Windows, try the (free) ActivePython distribution.
  2. Here is a very simple program to read a dicom file, modify a value, and write the result to a new dicom file:: 
        import dicom
    dataset = dicom.ReadFile("file1.dcm")
    dataset.PatientsName = 'anonymous'
    dicom.WriteFile("file2.dcm", dataset)
  3. See the files in the examples directory that came with this package, in particular the capture of an interactive session. This will quickly show you what you can do with this package.
  4. Use pydoc to access a kind of 'reference manual' for this package. See Documentation section below.

Documentation

pydoc

pydoc is a standard module in Python 2.1 and higher. Run it with a command line like: 
    python pydoc.py -w -g dicom
(The above command may need to be modified to "find" pydoc. For example, on my system: 
    python c:\python23\lib\pydoc.py -w -g dicom

This command will start a browser and allow you to surf through documentation of modules, classes and functions. The first page will contain all kinds of Python modules. Keep looking - the dicom link is there somewhere.

pydoc can also be accessed using the help function at the python prompt. For example: 

    >>>from dicom.dataset import Dataset
    >>>help(Dataset)

Learn the methods of the Dataset class; that is the one you will work with most directly.
